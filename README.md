# Análise de Dados de Vendas

Projeto de análise de dados com Python e SQL com o objetivo de extrair insights de vendas.

## Tecnologias
- Python (Pandas)
- SQL
- Matplotlib

## Análises realizadas
- Faturamento total
- Vendas por produto
- Vendas por mês
- Ranking de clientes

## Insights
- Identificação dos produtos mais vendidos
- Análise de desempenho mensal
- Clientes com maior volume de compras
